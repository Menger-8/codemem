# Memory in the Age of AI Agents: A Survey — Forms, Functions and Dynamics

> Yuyang Hu, Shichun Liu, Yanwei Yue, Guibin Zhang 等
> NUS / RUC / Fudan / PKU / NTU / UCSD / Oxford 等 | 2025 | 107页

## 📄 核心贡献

- 提出 **"形式-功能-动态"（Forms-Functions-Dynamics）三角框架**，统一分类LLM Agent记忆系统
- 超越传统的"短期/长期记忆"二分法，提出更细粒度的分类体系
- 系统梳理了记忆与LLM记忆、RAG、上下文工程的关系与区别
- 编译了全面的基准测试和开源框架资源

## 🔬 方法论（大白话讲解）

### 统一分类框架

#### 维度一：形式（Forms）—— 记忆用什么载体存储？

| 形式 | 子类 | 代表工作 | 类比 |
|------|------|----------|------|
| **Token级记忆** | 扁平(1D) | RAG、MemGPT | 便签纸 |
| | 平面(2D) | 知识图谱、Graphiti | 地图 |
| | 层次(3D) | LightMem、MemoryOS | 图书馆 |
| **参数记忆** | 内部参数 | MemoryLLM、M+ | 大脑突触 |
| | 外部参数 | Retroformer、Early Experience | 外接硬盘 |
| **潜在记忆** | 生成/复用/变换 | MemGen、LatentMemory | 潜意识 |

#### 维度二：功能（Functions）—— 为什么需要记忆？

| 功能 | 子类 | 说明 | 代表工作 |
|------|------|------|----------|
| **事实记忆** | 用户事实 | 记住用户偏好、历史 | Mem0、MemGPT |
| | 环境事实 | 记住环境状态、知识 | RAG、Graphiti |
| **经验记忆** | 基于案例 | 从过去案例中学习 | ExpeL、MemAgent |
| | 基于策略 | 提取可复用的策略 | MemSkill、LightSearcher |
| | 基于技能 | 学习可复用的技能 | SkillWeaver |
| | 混合 | 多种经验结合 | Mem-T |
| **工作记忆** | 单轮 | 当前对话/任务的上下文 | ReSum |
| | 多轮 | 跨会话的工作状态 | HiAgent |

#### 维度三：动态（Dynamics）—— 记忆如何运作？

| 阶段 | 说明 | 关键问题 |
|------|------|----------|
| **形成** | 从交互中提取记忆 | 什么值得记住？ |
| **演化** | 更新、合并、遗忘 | 信息如何保持最新？ |
| **检索** | 根据需要找回记忆 | 什么时候、找什么？ |

### 记忆生命周期

```
交互 → 形成(提取关键信息) → 演化(更新/合并/遗忘) → 检索(按需找回) → 辅助决策
```

这三阶段不一定同时发生，不同的调用模式产生不同的记忆效果：
- 仅在任务开始时检索一次 → 类似RAG
- 持续形成和演化 → 自进化记忆
- 仅在任务内积累 → 工作记忆

### 关键概念区分

| 概念 | Agent记忆 | LLM记忆 | RAG | 上下文工程 |
|------|-----------|---------|-----|-----------|
| **焦点** | 持久化认知状态 | 模型内部表示 | 静态知识访问 | 临时资源管理 |
| **演化** | 自我进化 | 架构优化 | 静态索引 | 动态组装 |
| **跨任务** | 支持 | 不支持 | 不支持 | 不支持 |

## 📊 实验结果（综述不包含原创实验）

综述的主要贡献是**分类和梳理**，而非原创实验。关键观察：

### 记忆系统演进时间线

```
2023: MemGPT (OS类比) → MemoryBank (遗忘曲线)
2024: A-Mem (Zettelkasten) → Mem0 (生产级) → HippoRAG (神经科学)
2025: Memory-R1 (RL) → Mem-α (RL) → AgeMem (统一管理) → MemSkill (技能进化)
      LightMem (人类记忆) → REMem (情景记忆) → MEM1 (恒定内存)
      Mem-T (树RL) → Memory in the Age of AI Agents (综述)
```

### 研究趋势

1. **从手工到自动**：早期系统依赖预定义规则，现在RL训练让Agent自主学习记忆策略
2. **从语义到情景**：从简单的事实存储发展到时间感知、空间感知的情景记忆
3. **从检索到生成**：从"检索已有记忆"到"生成新记忆"
4. **从单模态到多模态**：从纯文本扩展到图像、音频等
5. **从单Agent到多Agent**：从个体记忆到共享记忆

## ⚠️ 局限性

- **分类边界模糊**：某些系统可能同时属于多个类别
- **快速演进**：论文发表时可能已有新的重要工作
- **评估不统一**：不同论文使用不同基准和指标，难以直接对比
- **缺乏实验验证**：作为综述，不包含原创实验
